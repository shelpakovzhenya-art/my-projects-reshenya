# my projects reshenya

SEO-аудит и отчеты по проектам из задачи от 22.06.2026.

## Что внутри

- `reports-docx/` - итоговые Word-отчеты по каждому проекту.
- `reports-pdf/` - PDF-версии тех же отчетов, экспортированные через Microsoft Word.
- `render-check-png/` - PNG-рендеры страниц для визуальной проверки верстки отчетов.
- `scripts/create_seo_reports.py` - скрипт, которым были собраны DOCX-отчеты.
- `FIXES_APPLIED.md` - что удалось поправить и что осталось на стороне сервера/CMS.
- `VALIDATION.md` - как проверялись документы перед выгрузкой.
