# my projects reshenya

Репозиторий с результатами SEO-проверки проектов от 22.06.2026.

## Что было нужно сделать

Проверить проекты из таблицы доступов, найти SEO-проблемы, исправить то, что можно безопасно исправить через CMS, и оформить оставшиеся задачи в виде ТЗ.

## Откуда взяты данные

- таблица доступов Google Sheets;
- таблица SEO-материалов и курсов;
- публичная проверка сайтов;
- доступные CMS-админки;
- проверка страниц, robots.txt, sitemap.xml, редиректов и meta-тегов.

Яндекс.Вебмастер напрямую не проверен: был нужен вход в Яндекс ID.

## Где лежат файлы

- `reports-docx/` - Word-документы с ТЗ по каждому проекту;
- `reports-pdf/` - PDF-копии этих документов;
- `render-check-png/` - рендеры страниц для проверки, что документы нормально открываются;
- `scripts/create_seo_reports.py` - скрипт генерации DOCX.

## Что исправлено

- `www.kmel.ru`: обновлен meta description главной страницы.
- `aquaburmaster.ru`: запущена генерация sitemap.xml в CMS, но публичный sitemap не обновился. Задача оставлена в ТЗ.

## Что осталось сделать

Основные задачи вынесены в отдельные документы по проектам. В основном это:

- настроить 301-редиректы;
- закрыть дубли `/index.php`, `www/non-www`, `http/https`, слэш/без слэша;
- добавить canonical;
- исправить обработку URL-параметров;
- пересобрать sitemap.xml;
- после правок проверить сайты в Яндекс.Вебмастере.

Логины, пароли и исходные таблицы доступов в репозиторий не добавлены.
