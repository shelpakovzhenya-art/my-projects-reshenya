# my projects reshenya

SEO-ТЗ по проектам от 22.06.2026.

## Файлы

- `reports-docx/` - Word-файлы по каждому проекту.
- `reports-pdf/` - PDF-копии.
- `render-check-png/` - рендеры для проверки открытия документов.
- `reports-text/` - короткие текстовые проверки по отдельным проектам.
- `site-backups/` - бэкапы перед CMS-правками.

## Формат документов

В каждом файле только:

- `Сделано`
- `Что нужно сделать` - текстом, с пояснением `Зачем` по каждому пункту

Доступы, логины, пароли и исходные таблицы в репозиторий не добавлены.
