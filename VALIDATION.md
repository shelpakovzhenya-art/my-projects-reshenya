# Проверка файлов

- создано 6 DOCX-документов;
- документы открываются через `python-docx`;
- документы экспортированы в PDF через Microsoft Word;
- PDF отрендерены в PNG;
- страницы визуально проверены;
- добавлена отдельная текстовая проверка по Аквабурмастеру;
- сохранен бэкап редиректов Аквабурмастера перед favicon-правкой;
